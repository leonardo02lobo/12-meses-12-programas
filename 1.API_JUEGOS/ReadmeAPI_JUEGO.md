# API JUEGO

#### Este proyecto tiene como finalidad entender y aplicar el uso de las APIs, conocer sobre los conceptos de un ORM y aplicarlos.

#### Este proyecto fue desarrollado con las siguientes tecnologias

* HTML
* CSS
* Bootstrap
* Entity Framework
* SQL Server
